What Are Agentic Applications?

Unlike standard LLMs that produce single responses, agentic AI systems autonomously plan, decide, and execute multi-step tasks — creating a fundamentally new attack surface.

Autonomous Planning

Agents break complex goals into multi-step plans, selecting tools and data sources without human guidance at each step.

Tool & API Access

They invoke real-world tools — databases, APIs, file systems, code interpreters — amplifying the impact of any compromise.

Cascading Risk

A single manipulated input can propagate through multi-agent workflows, compounding into system-wide failures.

ASI01

Agent Goal Hijack

Attackers manipulate agent objectives through prompt injection, deceptive tool outputs, or poisoned data to redirect multi-step behavior.

→
ASI02

Tool Misuse & Exploitation

Agents apply legitimate tools in unintended ways — deleting data, over-invoking APIs — due to injection or misalignment.

→
ASI03

Identity & Privilege Abuse

Exploiting dynamic trust and delegation chains to escalate access via cached credentials or confused deputy scenarios.

→
ASI04

Supply Chain Vulnerabilities

Compromise of third-party models, tools, plugins, or agent personas loaded dynamically at runtime.

→
ASI05

Unexpected Code Execution

Adversaries exploit code-generation features to execute malicious scripts or binaries that compromise the host system.

→
ASI06

Memory & Context Poisoning

Corrupting RAG stores, embeddings, or long-term memory to bias future reasoning or plant persistent backdoors.

→
ASI07

Insecure Inter-Agent Comms

Intercepting, spoofing, or manipulating messages between agents due to weak authentication or integrity checks.

→
ASI08

Cascading Failures

A single fault propagating autonomously across agents and systems, compounding into widespread service failures.

→
ASI09

Human-Agent Trust Exploitation

Exploiting human over-reliance on AI to trick users into approving harmful actions or disclosing secrets.

→
ASI10

Rogue Agents

Agents that lose behavioral integrity — pursuing hidden goals, scheming to bypass safeguards, or self-replicating.
